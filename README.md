Run the appication in your local:

1) Clone the git repository https://github.com/pjagadish/ForumSite.git

2) bundle install

3) Set your db username and password

4) rake db:create

5) rake db:migrate

6) rake db:seed

7) rails s

Open Browser http://localhost:3000/
